# Hotel Management System

A simple console-based Hotel Management System built using Python.

## Features

- Book a room
- Cancel a booking
- Display available rooms
- View customer details
- Generate customer bills
- View room types and pricing
- Record check-in and check-out dates
- Search customers
- Display all bookings
- Add food charges
- Add extra service charges
- Save booking data to a file
- Load booking data from a file

## Technologies Used

- Python 3
- Dictionaries
- File Handling
- Object-Oriented Programming (Basic)

## How to Run

```bash
python main.py
```

## Note

This is one of my early Python projects. The goal was to practice Python by building a hotel management system with room booking, billing, customer management, and file handling. I plan to improve the code structure and add more features as I continue learning.
